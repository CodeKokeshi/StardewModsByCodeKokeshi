================================================================================
IMPLEMENTATION RULES
================================================================================

1. Ensure that the mod created or part of the mod created is compared to the Game's Code for 1-2-3 times.
2. Ensure that the latest build would not suffer from warnings or errors.
3. If the part of the mod is unimplementable then don't implement.
4. SET = scripts that take effect immediately (one-time action). Example: Instant Tree Growth grows currently planted trees. Does not run in background.
5. Toggle/Boolean = cheats that persist and behave depending on true/false state.
6. Follow the game's cap values for settings (except Health=620, Stamina=1100 as specified).
