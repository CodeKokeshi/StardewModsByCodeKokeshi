# Ultimate Player Cheats (StardewUI Edition)

A comprehensive player customization mod for Stardew Valley with a beautiful **tabbed UI** powered by [StardewUI Framework](https://github.com/focustense/StardewUI).

## Features

All cheats are organized into **10 easy-to-navigate tabs**:

| Tab | Features |
|-----|----------|
| **General** | Master toggle to enable/disable all cheats |
| **Movement** | Speed multiplier, added speed bonus, no clip, always run |
| **Health** | Infinite stamina/health, max overrides, regen rates |
| **Combat** | One-hit kill, always crit, damage/crit multipliers, attack/defense/immunity bonuses, no monsters |
| **Tools** | Tool area multiplier, no stamina cost, infinite water, axe/pickaxe power bonuses |
| **Items** | Magnetic radius multiplier/bonus, infinite items |
| **Skills** | XP multiplier, individual skill level overrides, luck settings |
| **Fishing** | Instant bite, instant catch, max quality, always treasure |
| **Economy** | Forage quality, sell/buy price multipliers, friendship multiplier, no friendship decay |
| **World** | Freeze time, freeze indoors, never pass out, animal happiness, crop/tree growth |

## Requirements

- [SMAPI](https://smapi.io/) 4.0+
- [StardewUI Framework](https://www.nexusmods.com/stardewvalley/mods/TODO) (required dependency)

## Installation

1. Install SMAPI and StardewUI Framework
2. Download this mod and extract to your `Mods` folder
3. Run the game!

## Usage

- Press **K** (configurable in config.json) to open the cheats menu
- Navigate tabs by clicking the icons at the top
- Toggle cheats with checkboxes, adjust values with sliders
- Click **Save & Close** to apply, or **Reset Defaults** to restore original settings
- Settings auto-save when closing the menu

## Configuration

The hotkey can be changed by editing `config.json`:
```json
{
  "OpenMenuKey": "K"
}
```

All other settings are configurable through the in-game UI.
