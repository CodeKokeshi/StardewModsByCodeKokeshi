## Requirements

- [SMAPI](https://smapi.io/) 4.0.0 or later
- [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) (recommended for in-game configuration)

## Installation

1. Install SMAPI and Generic Mod Config Menu
2. Download this mod
3. Extract to your `Stardew Valley/Mods` folder
4. Launch the game and configure via GMCM

## Configuration

Open the mod menu in-game (via GMCM) to adjust all settings. Each section is clearly labeled with emoji icons for easy navigation.

You can also edit `config.json` directly in the mod folder.

## Compatibility

- Works in single-player and should work in multiplayer (host only affects host)
- Compatible with most other mods
- Uses Harmony for patching, so conflicts are possible with mods that patch the same methods
