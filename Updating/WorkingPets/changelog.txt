1.4.0
- Exploration system with circular routes (clockwise/counter-clockwise, randomly chosen)
- Pet teleports directly to areas with forageables, pre-scans all areas first
- When area is cleared, pet teleports to next area in route
- Exploration ends when all areas cleared or inventory full, pet returns to farm
- Only one pet can explore at a time, explore option hidden when unavailable
- Pets can be whistled back during exploration (pauses, can resume via dialogue)
- Right-click pet to open command menu (only when stationary, not while walking)
- Each pet now has its own separate inventory (no longer shared)
- Forage quality based on player's foraging level (iridium/gold/silver chances)
- Pet sounds correctly match pet type (dogs bark, cats meow, turtles chirp)
- Only one pet can follow at a time (single follower enforcement)
- Whistle disabled for pets not in same area or when another pet is following
- Pet foraging while following now requires player to forage first
- Pet only forages nearby items (within 5 tiles) when player is within 8 tiles
- Foraging cancels immediately when player moves away or changes location
- All pets reset to idle state at start of each new day
- Auto-deposit matches item quality intelligently: prefers same quality, otherwise chooses highest quality in chest
- Auto-deposit attempts before inventory full error (pets continue working if space freed)
- Sounds only play when player is in same area as pet
- Instant farm scan for work detection (no more slow realization)
- Pet location shown with display name (e.g., "Pierre's General Store" not "SeedShop")
- Pet markers hidden during cutscenes, events, and dialogue
- Inventory size and auto-deposit fully configurable in GMCM
- EXTREME BLINK wall-pass system for obstacle navigation
- Complete follow system rewrite eliminates stuttering and flickering
- State machine: Idle, FollowingPlayer, Foraging, WallPassing
- Smooth velocity-based movement with proper acceleration/deceleration
- Follow distances: stop at 1.25 tiles, start at 2.5 tiles
- Speed scaling based on distance to player
- Various config options for notifications and behavior
- Full i18n support for all text

1.3.1

1.3.0
- Follow mode upgraded from experimental to fully functional
- "Follow Outside Farm" now enabled by default - pets follow everywhere
- Pets forage while following (automatically pick up berries, mushrooms, spring onions, etc.)
- HUD notification when pet picks up forageable item
- "No work to do" notification when no debris/trees/boulders found
- Stuck detection uses ultra-fast speed to bypass obstacles (no teleporting)
- Pets smoothly accelerate through walls when stuck, then slow down
- Message changed from "too far away" to "not on the same area" for clarity
- Added "Forage While Following" config option (enabled by default)
- Fast mode automatically deactivates when pet reaches player
- Whistle keybind system - press V (default, configurable) to open pet menu
- Whistle Key configurable in Generic Mod Config Menu
- Whistle menu shows full-body pet sprites with scrolling for many pets
- Menu shows pet name, status (Idle/Working/Following), disables unavailable pets
- Pet tracking arrows display pet icons (larger scale, center-biased)
- Pets in follow mode are passable (walk through them for doorways)
- Pets exit resting/sleeping when changing locations while following
- Trees at ALL growth stages (0-5) can be removed, including seeds
- Small trees (stages 0-4) use twig effects instead of full tree effects
- Fixed stuck loop counter not accumulating after warps
- Removed "(Experimental)" label from Follow Outside Farm option

1.2.0
- Multi-pet support! Each pet now works independently (Pet Licenses compatible).
- Target reservation system prevents pets from targeting the same debris.
- Collision avoidance keeps pets from stacking on each other.
- Pets now pause and stop moving when you talk to them.
- After 3 failed warp attempts, pets destroy target remotely (fixes infinite stuck loops).
- Trees at all growth stages (1-5) can now be removed, not just fully grown ones.
